# تطبيق قائمة مهام صوتية (Android) — بناء APK تلقائي عبر GitHub Actions

## ماذا يفعل التطبيق؟
- إدخال عنوان المهمة بالصوت (عربي)
- اختيار تاريخ/وقت للمهمة
- تذكير قبل الموعد بعدد ساعات (افتراضي 24 ساعة وقابل للتعديل)
- زر "تمت" (يلغي التذكير)
- حفظ المهام داخل الهاتف

## كيف تحصل على ملف APK بدون تثبيت Flutter؟
### 1) ارفع هذا المشروع إلى GitHub
1. ادخل GitHub وأنشئ Repository جديد (مثلاً: voice-tasks)
2. ارفع محتويات هذا الملف المضغوط كما هي (لا ترفع الملف المضغوط نفسه)
   - مهم: يجب أن يظهر مجلد `overlay/` ومجلد `.github/` في جذر المستودع.

### 2) شغّل البناء
1. افتح تبويب **Actions**
2. اختر Workflow باسم: **Build Android APK**
3. اضغط **Run workflow**

### 3) تحميل الـ APK
بعد انتهاء التشغيل:
1. افتح تشغيل الـ workflow
2. في الأسفل ستجد **Artifacts**
3. حمّل: `app-release-apk`
4. بداخلها يوجد: `app-release.apk`

## تثبيت APK على الجوال
- انقل `app-release.apk` إلى هاتفك
- افتحه
- فعّل "تثبيت من مصادر غير معروفة" عند الطلب
